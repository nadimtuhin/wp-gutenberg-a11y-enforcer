# Contributing to WP Gutenberg A11y Enforcer

Thank you for helping make the web more accessible!

## Getting Started

1. Fork the repo and clone locally.
2. Install dependencies: `npm install`
3. Activate a local WordPress install and symlink/copy the plugin.

## Development

- Follow [WordPress Coding Standards](https://developer.wordpress.org/coding-standards/).
- All new features must include unit tests.
- Run tests: `npm test` / `composer test`

## Pull Requests

- Branch from `main`.
- Keep PRs focused — one feature or fix per PR.
- Reference the related issue: `Closes #N`.
- Ensure CI passes before requesting review.

## Reporting Bugs

Use the [Bug Report](.github/ISSUE_TEMPLATE/bug_report.md) template.

## Accessibility Standards

This plugin enforces WCAG 2.1 AA. All contributions must themselves be accessible and must not weaken existing checks.
